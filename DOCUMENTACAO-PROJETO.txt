================================================================================
VISÃO GERAL DO PROJETO
================================================================================

Este é um monorepo Nx com um encurtador de links:

- Frontend Angular (apps/frontend) — interface para criar e listar links
- API NestJS (apps/api) — cria links, lista links ativos e redireciona
- PostgreSQL — persiste os links (via Docker localmente)
- Deploy — API no Render, frontend na Vercel

Fluxo principal: o usuário cola uma URL → a API gera um código de 6 caracteres
→ retorna shortUrl → ao acessar /api/r/:code, o servidor redireciona para a
URL original.

Fluxo:
  Usuário → Frontend Angular → POST/GET /api/links → NestJS API → PostgreSQL
  Visitante → GET /api/r/:code → NestJS API → 302 redirect → URL original

DTO de validação com class-validator:

--------------------------------------------------------------------------------
src/links/links.controller.ts
--------------------------------------------------------------------------------

create(dto)
  POST /api/links — recebe { url }, chama linksService.create(), retorna HTTP 201.

findAll()
  GET /api/links — lista links não expirados.


--------------------------------------------------------------------------------
src/links/redirect.controller.ts
--------------------------------------------------------------------------------

redirect(code, res)
  GET /api/r/:code — busca o link pelo código e responde com redirect HTTP 302
  para originalUrl.


--------------------------------------------------------------------------------
src/links/links.service.ts
--------------------------------------------------------------------------------

create(originalUrl)
  Normaliza a URL, gera código único, calcula expiresAt, salva no banco e
  retorna LinkResponse.

findAll()
  Busca links com expiresAt > agora, ordenados por createdAt DESC, mapeia para
  LinkResponse[].

findByCode(code)
  Busca por código; lança NotFoundException se não existir ou estiver expirado.

getExpiryDate() [privada]
  Calcula data de expiração com base em LINK_EXPIRY_DAYS (padrão 7 dias).

normalizeUrl(url) [privada]
  Remove espaços; adiciona https:// se não houver protocolo.

generateUniqueCode() [privada]
  Gera código aleatório de 6 caracteres alfanuméricos; tenta até 10 vezes
  garantir unicidade.

toResponse(link) [privada]
  Monta o objeto de resposta com shortUrl usando APP_URL + /r/:code.


--------------------------------------------------------------------------------
src/common/filters/http-exception.filter.ts
--------------------------------------------------------------------------------

catch(exception, host)
  Intercepta HttpException e padroniza a resposta JSON:
  { statusCode, message, error }.

================================================================================
COMO AS PEÇAS SE CONECTAM
================================================================================

1. Dev local:
   npm run db:up sobe o Postgres
   npm run start:api (porta 3000)
   npm run start (Angular na 4200, proxy /api → API)

2. Criar link:
   Angular POST /api/links
   → LinksController.create()
   → LinksService.create()
   → salva no banco
   → retorna shortUrl

3. Redirecionar:
   visitante acessa https://.../api/r/abc123
   → RedirectController.redirect()
   → findByCode()
   → HTTP 302

4. Produção:
   frontend na Vercel chama API no Render
   banco via DATABASE_URL
   CORS liberado para FRONTEND_URL
